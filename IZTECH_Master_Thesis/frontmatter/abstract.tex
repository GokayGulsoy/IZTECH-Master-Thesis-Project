\begin{abstractenv}
    The deployment of Large Language Models (LLMs) via Machine Learning as a Service (MLaaS) platforms raises critical data privacy concerns, as users must transmit sensitive information to untrusted servers. Fully Homomorphic Encryption (FHE) enables computation directly on encrypted data, but adapting Transformer architectures to the encrypted domain requires replacing all non-polynomial operations with low-degree polynomial approximations.

    This thesis presents the Learnable Polynomial Approximation Network (LPAN), a three-stage progressive framework that replaces \emph{all three} non-linear operations in BERT---GELU, Softmax, and LayerNorm---with learnable Chebyshev polynomial approximations. Key contributions include: (1)~distribution-weighted minimax polynomial initialisation that achieves $3.4\times$ lower error than standard Chebyshev; (2)~progressive layer-by-layer replacement with attention-level knowledge distillation; (3)~per-head polynomial Softmax enabling head specialisation; and (4)~the first complete polynomial LayerNorm replacement with co-adaptation. Evaluated on four BERT variants (Tiny through Base) on SST-2, LPAN achieves accuracy drops of only 0.11\%--0.34\% while replacing all three non-linear operations---outperforming existing methods (THE-X, Iron, MPCFormer, BOLT) that replace only two operations with 1--5\% accuracy drops. LPAN is the first framework enabling fully non-interactive encrypted Transformer inference.
\end{abstractenv}
